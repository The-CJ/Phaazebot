Token

Tokens are placeholder that get replaced with informations the the message is given out.

e.g.: 

Hello [user-mention], please read <#rules#>

Will be: 

Hello @NewUserName, please read #rules
Note: using <#rules#> will make a clickable channel mention,
 If you just use #rules it will be normal text.

Token list

[user-name] The User name
[user-mention] A @mention of the name
[channel-name] The name of the event channel of the name
[channel-mention] A #mention of the event channel
[server-name] Event server name
[member-count] Current event server member count

Special token list
<#name#> Channel mention of channel associated with name*
<#!id!#> Channel mention of channel associated with id*
* = if none found replace with: (unknown)

Blacklist

Pretty easy to unserstand, if any word of the blacklist is found in the message, execute the punishment

Note:
Every word and or phase is case insensitive
Means:
The word "ass" gets found in "I play a brass instrument"
and by this will get a punishment

Blacklist

By default every link gets punished, but here you can define regular expressions that are allowed.

e.g:
If you ban links, but want to allow all google links, including all subdomains.
You add this: ^.*google.com$
