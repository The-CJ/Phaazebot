Master Controlls

All options here will have server wide impact, that's why only the server owner can change stuff here.

Disable Level system

If this is enabled, the entire level system is server wide disabled.
Means no level-up messages, no EXP gain, no currency actions and also,
no commands that use the level system. Commands like: "Statistics" or "Leaderboard"
(The serverowner will always be able to use all commands)

Disable all Normal-Commands

If this is enabled, all commands with a requirement of 'Everyone' are disabled.
However, commands that require 'Regular' or 'Mod' will still work.
(The serverowner will always be able to use all commands)

Disable all Regular-Commands

If this is enabled, all commands with a requirement of 'Regular' are disabled.
However, commands that require 'Everyone' or 'Mod' will still work.
(The serverowner will always be able to use all commands)

Disable all Mod-Commands

If this is enabled, all commands with a requirement of 'Mod' are disabled.
However, commands that require 'Everyone' or 'Regular' will still work.
(The serverowner will always be able to use all commands)
