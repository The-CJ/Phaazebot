PhaazeDiscord - Level System

Intro

PhaazeDiscord comes with a level system to track who is the most active one you the server and also announce level up's
(EXCEPT : The level system is disabled in a channel by a moderator or server wide by the owner)

How it works

You gain 1 EXP with every message, every random (5, 10) sec.

This way you can't spam your way to victory... i mean you can, but not that fast
And im sure someone will stop you, if not... then not.

How to disable

It can be disabled for each channel, or server wide.

  • Channel wide: Moderator or higher can use: >>settings level to dis/en-able levels

  • Server wide: Server owner can use: >>>master level to disable it in all channels

Members can't use >level or >leaderboard if levels are disabled.

Editing Level

Levels can be edited by mods using >>level but
edited members will be marked by a [EDITED] mark in all level stats.

To reset this you have to set the level back to 0 and start fresh.
