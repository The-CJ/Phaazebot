PhaazeDiscord - Custom Commands

Intro

The Normal commands are not enough?

Make your own!

You can add up to 150 Custom commands, without trigger limits.

Want a bot that greets everyone saying "Hi" or "Hey" or whatever, just make it. There are no Syntax rules.
(NOTE: "@everyone" as a trigger is not forbidden, but stupid)

It also supports command Token e.g.: [uses] - will be replaced with the number of times the command was triggered.

How it works

Moderator or higher can create or delete commands with >>addcom and >>delcom,
after you set some, everyone can use >custom to check what commands are available.
Commands can also be viewed in web on phaaze.net/discord/custom/<serverID>

Moderator and higher can also visit the dashboard at phaaze.net/discord/dashboard/<serverID>#custom
and can manage commands from there.

All custom commands can be used in all channels,
but moderators or higher can disable the use per channel with >>settings custom
