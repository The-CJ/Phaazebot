PhaazeDiscord - Commands - Doujin

Intro

Thanks go out to Tsuminos Admin, who added an API on my request for this service.
With this Command you can request doujins by a search criteria.

Content

  • Restrictions
  • Usage
  • Terms
  • Example

Restrictions

  • Not via DM
  • Extended cooldown: 5sec
  • Must be enabled for channels via: >>settings nsfw before using

Usage

>doujin [term] (more terms) ....

Note

If you want to make a " " (blank space), replace it with "_" (unserscore)

e.g.: Gurren Lagann -> gurren_lagann

Terms

  • Random search

    <r  (only r, withour any other options)

    Returns a complete random Doujin

  • Include tags

    <i

    Include one or multiple tags to the search - separated by " " (blank space)

    (No Autocomplete, don't make typos)

  • Exclude tags

    <e

    Exclude one or multiple tags to the search - separated by " " (blank space)

    (No Autocomplete, don't make typos)

  • Anime search

    <a or <anime

    Search one or multible animes - separated by " " (blank space)

    (Trys to Autocomplete: "nisekoi" --> "Nisekoi / Ã£Æ’â€¹Ã£â€šÂ»Ã£â€šÂ³Ã£â€šÂ¤")

  • Character search

    <c or <char

    Search one or multible characters - separated by " " (blank space)

    (Try to Autocomplete: "golden_darkness" --> "Golden Darkness / Konjiki no Yami")

  • Star search

    <s or <star

    The Minimum Star Rating (1-5)

    (Numbers only)

  • Page Maximum Search

    <pmax or <page_maximum

    The Maximum Site Count

    (Numbers only)

  • Page Minimum Search

    <pmin or <page_minimum

    The Minimum Site Count

    (Numbers only)

  • Rawsearch

    <=

    Rawsearch for entrys, must be last option/term.

    (Without "_" (underscore), just use normal " " (blank space))
