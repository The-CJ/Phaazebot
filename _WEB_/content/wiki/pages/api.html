To PhaazeAPI


All (or at least, most) moduls and features of the PhaazeOS can be accessed via a open API,
so you can access stuff like Discord server commands, twitch user level, or even like useless stuff
like saved object hash values in PhaazeDB - Fileserver, or even more.

Developers are welcome to create new cool tools with the API or just use it for your own (Discord/Twitch/etc.) Bots
to provide your server member with annoying usfull live updates... or worse.
I'm sure, you will find something.
