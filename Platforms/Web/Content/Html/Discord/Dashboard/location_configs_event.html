Automated Events

A new member joined? Great, flood them with welcome messages, rule notices and give them a role that blocked them from doing anything until they accepted. (Or don't and just say hello)

Welcome message

Send a message to a predefined channel, whenever a new user joins.
If no channel is set or the message is empty, no message will be sent.
This message supports tokens.

Private DM welcome message

A message that is sent to a user via DM,
it gets sent at the same time as the "Welcome message" just via DM.
Please keep in mind that this message may not reach the user, due to there privacy settings.
This message supports tokens.

Autorole

The selected role will be given to every user, that joins the server.
This must be a role Phaaze has permission to assign or empty.

Leave message

Send a message to a predefined channel, whenever a user leaves.
If no channel is set or the message is empty, no message will be sent.
This message supports tokens.
