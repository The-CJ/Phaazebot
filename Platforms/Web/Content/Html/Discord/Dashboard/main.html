Token

Tokens are placeholder that get replaced with informations if the message content is given out.

e.g.:

Hello [user-mention], please read <#rules#>. Have a nice day :D

will be:

Hello @NewUserName, please read #rules. Have a nice day :D
Note: using <#rules#> will make a clickable channel mention
If you just use "#rules", it will be normal text.

Token List

[user-name]
The user name
[user-mention]
A @mention of the user
[channel-name]
Name of the event channel
[channel-mention]
A #mention of the event channel
[server-name]
The event server name
[member-count]
Current event server member count
[lvl]
The users level
[exp]
The users exp amount
[rank]
The users ranked based on exp

Special token List

<#name#>
#mention of channel associated with name*
<#!id!#>
#mention of channel associated with id*
* = If none is found, it's replaced with: (unknown)
